# Данный проект является выпускной квалификационной работой и представляет собой информационную систему, разработанную для отдела сбыта металлургического завода
Данная информационная система:
- отображает актуальную информацию о предприятии, производимой продукции и оказываемых услугах;
- предоставляет возможность потенциальным заказчикам подавать заявки в электронном виде;
- позволяет создавать, заключать и хранить договора в электронном виде;
- предоставляет сотрудникам интерфейс для редактирования информации разделов товаров и услуг;
- предоставляет сотрудникам интерфейс для обработки заявок и договоров;
- предоставляет администратору интерфейс для редактирования разделов и управления пользователями.
- включает внутреннюю почту для обмена сообщениями и получения уведомлений.

Проект выполняет все поставленные задачи. Результат работы системы можно посмотреть по ссылке - https://youtu.be/b1BwMcTuHKI
Информационная система размещена по ссылке - https://blc-centrolite.ru/

Однако, в связи с коротким сроком на выполнение работы приношу читателю свои извинения за:
- места, где оформление не соответствует стандартам PSR;
- возможное наличие "мусора" среди строк файлов;
- повторение кода;
- незавершенность проекта (некоторые разделы пусты или не имеют финальной логики);
- отсутствие адаптивной верстки;
- отсутствие правильной иерархии папок и файлов.
